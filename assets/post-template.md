Use this exact format when presenting the final post to the user.

---

---

## Rules for Filling the Template

1. **Both versions are required.** Always provide Standard and Punchier.
2. **Character counts must be accurate.** Count the actual characters, not an estimate.
3. **AI slop score:** Count how many patterns from `references/anti-slop-patterns.md` were found in the FINAL draft (after cleanup). 0-1 = score 0-1. 2-3 = score 2-4. 4+ = rewrite.
4. **Voice match:** Compare 5 dimensions (tone, sentence length, structure, vocabulary, formatting) against the voice profile. Each dimension is 20%. Report the match percentage.
5. **Sources:** List every source used for data points. Include year. If a source was only partially accessible, note it.
6. **Posting guide:** Base "best time" on the user's audience timezone. If unknown, default to Tuesday or Thursday 10-11 AM in the user's local time.
7. **Next post:** Suggest a topic that naturally follows. If this post was contrarian, suggest a "how-to" follow-up. If this was data-driven, suggest a personal experiment follow-up.
